Como funciona o gerador de links

Encontre uma oferta de produto;

Copie a URL do site do produto escolhido;

Cole a URL no campo “cole abaixo a URL que deseja divulgar”;

Clique em “gerar URL”;

No campo abaixo será disponibilizado o seu link exclusivo para divulgação.

Caso queira transformar a URL em um link mais curto clique em “converter para URL curta”;

Você será direcionado automaticamente para o site www.bitly.com onde você poderá se cadastrar e gerar uma URL curta.
